# Take U Forward Calendar

A lightweight calendar and consistency-tracking app built with React and Vite.

## Tech Choices

- React 19 for the UI component model and state management.
- Vite for fast local development and production builds.
- date-fns for month/date calculations.
- Plain CSS for the layout, animations, and custom visual styling.

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```

## Notes

- Run `npm run lint` to check the codebase with ESLint.
- The app stores month notes and tree progress in localStorage.
